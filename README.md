# PhISCS

PhISCS is a tool for sub-perfect tumor phylogeny reconstruction via integrative use of Single Cell and Bulk Sequencing data.

## Installation
```
git clone https://github.com/haghshenas/PhISCS.git
cd PhISCS
./configure
```
